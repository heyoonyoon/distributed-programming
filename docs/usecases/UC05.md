# UC05. 의료보험을 청구한다

**액터:** 보험가입자

---

## 선행 조건

- 보험가입자가 시스템에 로그인한 상태여야 한다.
- 유효한 의료보험 계약이 존재해야 한다.

## 후행 조건

- 청구 정보가 저장된다.
- 간단한 청구의 경우: 보험금 지급 처리가 즉시 진행된다.
- 복잡한 청구의 경우: [확장 유스케이스: UC14. 담당자를 지정한다] 가 수행되어 담당자가 배정된다.

---

## 베이직 플로우

1. 보험가입자가 메인 화면에서 '의료보험 청구' 메뉴를 클릭한다.
2. 시스템은 청구 정보 입력란(청구 사유, 진료일, 병원명, 청구 금액)을 출력한다.
3. 보험가입자가 청구 정보를 입력하고 진료비 영수증 등 증빙 서류를 첨부한다. (E1)
4. 보험가입자가 '청구 신청' 버튼을 클릭한다.
5. 시스템은 청구 정보를 저장하고, 청구 내용의 복잡도를 자동으로 판별한다.
6. 시스템이 '간단한 청구'로 판별한 경우, [UC17. 보험금을 지급한다]를 즉시 수행한다. (A1)
7. 시스템은 보험가입자에게 청구 처리 결과를 이메일/문자로 통보한다.

---

## 대안 흐름

### A1. 복잡한 청구로 판별된 경우 (5단계 분기)

- **A1-1.** 시스템은 해당 청구 건을 '심사 대기' 상태로 변경한다.
- **A1-2.** [확장 유스케이스: UC14. 담당자를 지정한다] 를 수행하여 담당 직원을 배정한다.
- **A1-3.** 시스템은 보험가입자에게 "담당자 배정 후 심사를 진행합니다." 안내를 발송한다.

---

## 예외 흐름

### E1. 첨부 파일 형식 오류 (3단계에서 발생)

- **E1-1.** 시스템은 "지원하지 않는 파일 형식입니다. (허용: PDF, JPG, PNG)" 메시지를 출력한다.
