# UC15. 자동차 사고 이력을 조회한다

**액터:** 금융감독원 (외부 기관 시스템)

---

## 관계

- [UC13. 보험 가입을 심사한다] 중 자동차보험 심사 건에 한해 수행됨

## 선행 조건

- [UC13. 보험 가입을 심사한다]에서 자동차보험 가입 심사 정보가 열람된 상태여야 한다.
- 외부 기관(금융감독원) 정보 연동 기능이 정상 상태여야 한다.

## 후행 조건

조회된 사고 이력 정보(사고 건수, 유형, 지급액 합계, 면허 상태)가 심사 화면에 출력된다.

---

## 베이직 플로우

1. 시스템은 심사 대상이 자동차보험 건임을 확인하고, 외부 기관 정보 조회를 자동으로 시작한다.
2. 시스템은 가입 신청자의 주민등록번호를 기반으로 외부 기관에 사고 이력 조회를 요청한다. (E1)
3. 외부 기관 시스템이 사고 이력 정보를 전달한다.
4. 시스템은 전달받은 정보(최근 3년간 사고 건수, 사고 유형, 보험금 지급 총액, 현재 면허 상태)를 정리하여 화면에 출력한다.
5. [UC13. 보험 가입을 심사한다]의 6단계로 돌아간다.

---

## 예외 흐름

### E1. 정보 연동 실패 (2단계에서 발생)

- **E1-1.** 외부 시스템 응답 지연 또는 오류 시 시스템은 "정보 조회 실패" 안내창을 띄운다.
- **E1-2.** 보험사 직원이 '재시도' 버튼을 클릭하거나 '직접 입력 전환'을 선택한다.
- **E1-3.** 직접 입력 전환 선택 시 보험사 직원이 사고 이력 정보를 직접 입력할 수 있는 화면이 출력된다.
