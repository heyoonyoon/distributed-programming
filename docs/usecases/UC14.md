# UC14. 담당자를 지정한다

**액터:** 보험사 직원 (시스템에 의한 자동 배정 또는 수동 배정)

---

## 관계

- [UC12. 보험금 지급을 심사한다] 에 의해 반드시 수행됨
- [UC05. 의료보험을 청구한다] 에서 복잡한 청구 건에 한해 수행됨

## 선행 조건

- 심사 대기 상태의 보험금 지급 청구 건이 존재해야 한다.
- 담당 가능한 보험사 직원이 등록되어 있어야 한다.

## 후행 조건

- 해당 청구 건에 담당자가 배정되어 저장된다.
- 배정된 담당자에게 알림이 발송된다.

---

## 베이직 플로우

1. 시스템은 심사 대기 상태의 청구 건에 대해 담당자 배정을 시작한다.
2. 시스템은 현재 업무량, 전문 분야, 근무 상태를 기준으로 적합한 담당자를 자동으로 선정한다. (A1)
3. 시스템은 선정된 담당자를 해당 청구 건에 연결하여 저장한다.
4. 시스템은 배정된 담당자에게 "새 심사 건이 배정되었습니다." 알림을 발송한다.
5. 시스템은 청구 건 상세 화면에 담당자 정보를 출력한다.

---

## 대안 흐름

### A1. 수동 배정 (2단계 분기)

- **A1-1.** 관리자 또는 상위 직원이 '수동 배정' 버튼을 클릭한다.
- **A1-2.** 시스템은 배정 가능한 직원 목록(이름, 현재 담당 업무 수, 전문 분야)을 출력한다.
- **A1-3.** 관리자가 담당자를 선택하고 '배정 확정' 버튼을 클릭한다.
- **A1-4.** 3단계로 돌아간다.

---

## 예외 흐름

### E1. 배정 가능한 담당자 없음 (2단계에서 발생)

- **E1-1.** 시스템은 관리자에게 "배정 가능한 담당자가 없습니다." 알림을 발송한다.
- **E1-2.** 관리자가 업무를 다시 나눈 후 수동으로 배정을 진행한다.
