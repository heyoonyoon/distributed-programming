# UC17. 보험금을 지급한다

**액터:** 없음 (시스템이 자동으로 수행)

---

## 관계

- [UC05. 의료보험을 청구한다]의 간단한 청구 시 수행됨
- [UC12. 보험금 지급을 심사한다]에서 승인된 경우 수행됨

## 선행 조건

- 간단한 청구이거나 심사 결과 '승인'이 내려진 상태여야 한다.
- 보험가입자의 받을 계좌 정보가 등록되어 있어야 한다.

## 후행 조건

- 보험금 지급 내역이 기록된다.
- 보험가입자의 계좌로 보험금이 송금된다.
- 보험가입자에게 지급 완료 안내가 발송된다.

---

## 베이직 플로우

1. 시스템은 승인된 청구 건의 지급 금액과 받을 계좌 정보를 확인한다.
2. 시스템은 금융사 연동망을 통해 해당 계좌로 보험금 송금을 요청한다. (E1)
3. 송금이 완료되면 시스템은 지급 정보(금액, 일시, 계좌)를 저장한다.
4. 시스템은 보험가입자에게 "보험금 [지급 금액]원이 지급되었습니다." 안내를 이메일/문자로 발송한다.
5. 해당 청구 건의 상태를 '처리 완료'로 변경한다.

---

## 예외 흐름

### E1. 송금 실패 (2단계에서 발생)

- **E1-1.** 계좌 번호 오류나 은행 시스템 문제로 송금이 실패하면 시스템은 "지급 실패" 상태로 기록한다.
- **E1-2.** 시스템은 보험사 직원에게 송금 실패 알림을 보낸다.
- **E1-3.** 보험사 직원이 가입자에게 계좌 정보를 다시 확인하고, 확인 후 다시 송금을 시도한다.
