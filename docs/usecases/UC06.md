# UC06. 개인 정보를 수정한다

**액터:** 보험가입자

---

## 선행 조건

- 보험가입자가 시스템에 로그인한 상태여야 한다.
- 본인 인증이 완료된 상태여야 한다.

## 후행 조건

변경된 개인정보가 반영되며, 변경 이력이 기록된다.

---

## 베이직 플로우

1. 보험가입자가 마이페이지에서 '개인정보 수정' 메뉴를 클릭한다.
2. 시스템은 본인 인증 화면(휴대폰 인증 또는 공동인증서 등)을 출력한다.
3. 보험가입자가 본인 인증을 완료한다. (E1)
4. 시스템은 현재 등록된 개인정보(연락처, 주소, 이메일, 계좌번호)를 출력한다.
5. 보험가입자가 수정할 항목을 변경하고 '저장' 버튼을 클릭한다. (E2)
6. 시스템은 변경된 정보를 저장하고 변경 기록을 남긴다.
7. 시스템은 "개인정보가 성공적으로 변경되었습니다." 메시지를 출력하고, 등록된 연락처로 변경 완료 알림을 발송한다.

---

## 예외 흐름

### E1. 본인 인증 실패 (3단계에서 발생)

- **E1-1.** 시스템은 "본인 인증에 실패하였습니다. 다시 시도해 주세요." 메시지를 출력한다.
- **E1-2.** 3회 연속 실패 시 해당 기능 접근을 10분간 차단한다.

### E2. 저장 실패 (5단계에서 발생)

- **E2-1.** 시스템은 "저장에 실패하였습니다. 잠시 후 다시 시도해 주세요." 메시지를 출력하고, 입력된 데이터를 유지한다.
