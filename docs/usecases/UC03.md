# UC03. 보상 처리 현황을 확인한다
- **액터**: 보험가입자
- **선행 조건**:
    - 보험가입자가 시스템에 로그인한 상태여야 한다.
    - 진행 중인 보상 처리 건이 하나 이상 존재해야 한다.
- **후행 조건**: 현재 보상 처리 진행 상황이 화면에 출력된다.

### 베이직 플로우
1. 보험가입자가 메인 화면에서 '보상 처리 현황' 메뉴를 클릭한다.
2. 시스템은 진행 중인 보상 처리 목록(접수번호, 접수일, 보험 종류, 현재 상태)을 출력한다. (A1)
3. 보험가입자가 특정 건의 '상세보기' 버튼을 클릭한다.
4. 시스템은 해당 건의 처리 단계별 진행 현황(접수 완료 → 담당자 배정 → 심사 중 → 완료)을 시간 순서에 따른 흐름도로 출력한다.
5. 시스템은 현재 단계의 예상 처리 기간 및 담당자 연락처를 함께 출력한다.

### 대안 흐름
**A1. 진행 중인 보상 건 없음 (2 단계 분기)**
- A1-1. 시스템은 "현재 진행 중인 보상 처리 건이 없습니다." 메시지를 출력한다.
