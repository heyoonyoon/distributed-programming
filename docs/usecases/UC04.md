# UC04. 보상 이력을 조회한다

**액터:** 보험가입자

---

## 선행 조건

- 보험가입자가 시스템에 로그인한 상태여야 한다.

## 후행 조건

보험가입자의 보상 이력 목록이 화면에 출력된다.

---

## 베이직 플로우

1. 보험가입자가 메인 화면에서 '보상 이력 조회' 메뉴를 클릭한다.
2. 시스템은 조회 기간 선택 화면(기본값: 최근 1년)을 출력한다.
3. 보험가입자가 조회 기간을 설정하고 '조회' 버튼을 클릭한다.
4. 시스템은 해당 기간의 보상 이력 목록(처리일, 보험 종류, 청구 금액, 지급 금액, 처리 결과)을 출력한다. (A1)
5. 보험가입자가 특정 이력의 '상세보기' 버튼을 클릭한다.
6. 시스템은 해당 이력의 상세 내용(청구 사유, 제출 서류, 심사 결과, 지급 내역)을 출력한다.

---

## 대안 흐름

### A1. 이력 없음 (4단계 분기)

- **A1-1.** 시스템은 "해당 기간의 보상 이력이 없습니다." 메시지를 출력한다.
