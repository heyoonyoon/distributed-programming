## UC01. 보험 상품을 조회한다
- **액터**: 보험가입자
- **선행 조건**: 사용자가 시스템에 접속한 상태여야 한다. (비로그인 상태에서도 조회 가능)
- **후행 조건**: 조회된 보험 상품 목록 및 상세 정보가 화면에 출력된다.

### 베이직 플로우
1. 보험가입자가 메인 화면에서 '보험 상품 조회' 메뉴를 클릭한다.
2. 시스템은 보험 종류 선택 화면(의료보험 / 자동차보험)을 출력한다.
3. 보험가입자가 조회할 보험 종류를 선택한다. (E1)
4. 시스템은 해당 종류의 보험 상품 목록(상품명, 보장 내용 요약, 월 보험료 예시)을 화면에 출력한다.
5. 보험가입자가 특정 상품의 '상세보기' 버튼을 클릭한다.
6. 시스템은 해당 상품의 상세 정보(보장 항목, 보장 금액, 면책 사항, 보험료 산출 기준)를 화면에 출력한다.

### 대안 흐름
**A1. 검색 필터 사용 (4 단계 분기)**
- A1-1. 보험가입자가 필터 옵션(월 보험료 범위, 보장 항목 키워드)을 입력한다.
- A1-2. 시스템은 필터 조건에 맞는 상품 목록을 갱신하여 출력한다.

### 예외 흐름
**E1. 조회 결과 없음 (3 단계에서 발생)**
- E1-1. 시스템은 "해당 조건에 맞는 보험 상품이 없습니다." 메시지를 출력한다.
