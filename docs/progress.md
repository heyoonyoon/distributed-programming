## include / extend 관계 (총 4개)

| 종류 | 기본 유스케이스 | 대상 유스케이스 | 조건 |
|------|--------------|--------------|------|
| include | UC12 보험금 지급을 심사한다 | UC14 담당자를 지정한다 | 항상 (필수) |
| extend | UC05 의료보험을 청구한다 | UC14 담당자를 지정한다 | 복잡한 건으로 판별된 경우 |
| extend | UC13 보험 가입을 심사한다 | UC15 자동차 사고 이력을 조회한다 | 자동차보험 건인 경우 |
| extend | UC12 보험금 지급을 심사한다 | UC17 보험금을 지급한다 | 심사 승인 시 |

> include = 항상 실행 / extend = 조건부 실행
